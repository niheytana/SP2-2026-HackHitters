Project Explanation:
Our project predicts IPL Powerplay scores using historical IPL match data.

The model uses:
* Venue averages
* Batting team statistics
* Batsman performance
* Bowler impact

These features are combined using weighted averages to predict Powerplay scores for IPL innings.
